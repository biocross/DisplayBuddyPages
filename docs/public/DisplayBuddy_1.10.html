DisplayBuddy 1.10

New Features

  • Preset Rename: You can now rename your presets.

Improvements

  • Fixes an issue where the enable DDC read checkbox in preferences would automatically get disabled even if the read succeeds.
  • Fixes an issue where the more button would show when you have just 3 presets.
  • Fixes an issue where presets would sometimes not work after restarting your Mac. It's highly recommended you recreate your presets so that DisplayBuddy can create the required metadata for the fix.

Previous Releases

  • Presets UI: You can now view and use more than 4 presets.
  • Display Sync: You can now sync controls like brightness and contrast across multiple displays. To get started, click on the Display Name > Sync to this display.
  • Display Modes: You can now view detailed information about the different display modes DisplayBuddy uses to control your display. Click on the Display Name > What do these mean?
  • Media Keys: You can now use the brightness keys on your keyboard to control brightness! You can turn on this feature in Preferences > Keyboard Shortcuts.
