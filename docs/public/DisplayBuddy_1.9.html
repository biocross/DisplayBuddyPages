DisplayBuddy 1.9

New Features

Improvements & Fixes

  • DisplayBuddy now works much better when you're using hubs or docks to connect to your display.
  • The menu bar icon plays much better with light and dark modes in macOS.
  • Improved About page in Preferences.

Previous Releases

  • Display Sync: You can now sync controls like brightness and contrast across multiple displays. To get started, click on the Display Name > Sync to this display.
  • Display Modes: You can now view detailed information about the different display modes DisplayBuddy uses to control your display. Click on the Display Name > What do these mean?
  • You can now use the brightness keys on your keyboard to control brightness! You can turn on this feature in Preferences > Keyboard Shortcuts.
