Questions? Answers.

Will it work with my display?

DisplayBuddy talks to your monitor using the DDC spec. If you've a monitor that was manufactured in the last 5 years or so, there's a very high chance it will work great with DisplayBuddy!

How does it work?

The app uses Apple's IOKit framework to communicate with external displays using the DDC spec. It uses the ddcutil C package for linux as base and builds on top that for macOS.

Do you offer a free trial?

Is this a Native app?

Yep, 100% Native, written in Swift using AppKit to support macOS 10.11 & above.
